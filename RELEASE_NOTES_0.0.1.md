# MagicBattery 0.0.1 Release Notes

## 中文版

MagicBattery `0.0.1` 是项目的首个可运行版本，完成了菜单栏电池监控产品的基础形态搭建。

本版本重点提供以下能力：

### 核心功能

- 实时查看 Mac 本机电池状态
- 查看常见蓝牙设备电量（键盘、鼠标、耳机、AirPods 等）
- 通过菜单栏快速访问电池信息
- 支持低电量通知提醒
- 提供多尺寸 Widget 展示

### 适用场景

- 日常查看 Mac 与常用外设电量
- 在设备低电量前及时收到提醒
- 通过桌面 Widget 或菜单栏快速掌握当前设备状态

### 已知限制

- 部分蓝牙设备的电量数据可能不完整，或系统无法提供
- AirPods 单耳与充电盒电量细节支持仍有限
- 当前版本以“先可用”为目标，后续会继续提升准确性与展示体验

---

## English

MagicBattery `0.0.1` is the first usable release of the project.
It establishes the core menu bar battery monitoring experience for Mac and supported devices.

### Highlights

- Real-time Mac battery monitoring
- Bluetooth battery monitoring for common peripherals and audio devices
- Quick battery access from the menu bar
- Low battery notifications
- Multi-size widget support

### Suitable Use Cases

- Checking battery status for Mac and connected accessories
- Getting notified before devices run low on power
- Viewing battery status quickly from widgets or the menu bar

### Known Limitations

- Some Bluetooth battery values may be incomplete or unavailable
- AirPods earbud / charging case detail support is still limited
- This release focuses on delivering the first usable product baseline, with more refinements planned later
